# ReSpy
Respy is a pentesting ([Penetration Testing](https://en.wikipedia.org/wiki/Penetration_test)) tool for roblox. It is mainly focused to work on 
executor enviroments, and to work on every one.

A Remote spy is mainly a pentesting tool for roblox, but can be seen on other games.

The term "spy" is beacuse its intercepting server-to-client and client-to-server calls.

This tool was made by LenkaToJaMarcel, but rather likely to be known as blocky.
