Devlog

A running changelog of every Respy update — what changed, what broke, and what got fixed.

Jun 2025
Latest
v1.4.0 Multi-executor support & UI overhaul
Rewrote the hook layer to work across Solara, Xeno, and other low-UNC executors without needing full metatable access. The UI was rebuilt with a scale-only layout so it fits any resolution cleanly.
Added executor detection with automatic hook strategy selection
Rebuilt UI using scale-only sizing — no more off-screen panels on 4K
hookmetamethod fallback for executors without full UNC support
Fixed remote list scroll snapping on large games
feature fix refactor
Apr 2025
2 months ago
v1.3.2 Hotfix — Roblox engine update compatibility
Roblox's April engine update changed how RemoteFunction callbacks are stored internally, breaking capture on return values. This patch restores full bidirectional capture.
Patched RemoteFunction return value capture after engine update
Fixed nil argument serialization causing output panel crash
fix
Mar 2025
3 months ago
v1.3.0 Argument inspector & filter system
Added a tree-view argument inspector so you can expand tables and instances inline. Also ships a name-based filter so you can suppress noisy remotes while watching the ones you care about.
Tree-view inspector for nested table and Instance arguments
Per-remote whitelist / blacklist filter with regex support
Copy-to-clipboard button on every captured call
Hook overhead reduced ~40% via call batching
feature perf
Jan 2025
5 months ago
v1.2.0 Loadstring distribution & open source release
Respy goes fully open source. Source and loadstring builds are now hosted on GitHub and linked from the site. Anyone can audit, fork, or submit PRs.
Published full source to GitHub under MIT license
Separate loadstring build for one-line injection
Added CONTRIBUTING.md and issue templates
feature
Nov 2024

v1.0.0 Initial release
First public release of Respy. Basic RemoteEvent and RemoteFunction interception using hookmetamethod, with a minimal floating UI panel.
RemoteEvent FireServer / OnClientEvent interception
RemoteFunction InvokeServer capture
Floating output panel with scrollable log
